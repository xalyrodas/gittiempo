# Comando Descripción
1 git clone Crea una copia del repositorio
2 git status Muestra el estado de los archivos en el directorio de trabajo y el área de preparación
3 git add Añade archivos al área de preparación
4 git commit Confirma los cambios añadidos al área de preparación
5 git push Sube los commits al repositorio remoto
6 git pull Actualiza el repositorio local con los cambios del repositorio remoto
7 git branch Lista, crea o elimina ramas
8 git checkout Cambia de rama o restaura archivos en el área de trabajo
9 git merge Combina dos ramas
